# Radio (spec)

# Notes

- Single radio component.
- Will contain Label.
- Will have to provide some type of prop to configure the color of the selected radio button.

| Attributes  |  Values  | Optional ? |
| :---------- | :------: | ---------: |
| `id`        |  string  |        Yes |
| `name`      |  string  |         No |
| `value`     |  string  |         No |
| `label`     |  string  |         No |
| `isChecked` | boolean  |         No |
| `onChange`  | function |         No |
